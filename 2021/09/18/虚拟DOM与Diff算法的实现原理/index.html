иҷҡжӢҹDOMдёҺDiffз®—жі•зҡ„е®һзҺ°еҺҹзҗҶ

еҸ‘иЎЁдәҺ 2021-09-18 йҳ…иҜ»ж¬Ўж•°пјҡ йҳ…иҜ»ж¬Ўж•°пјҡ Valineпјҡ
жң¬ж–Үеӯ—ж•°пјҡ 2.4k йҳ…иҜ»ж—¶й•ҝ ≈ 2 еҲҶй’ҹ

еүҚиЁҖ

Vue жәҗз ҒдёӯиҷҡжӢҹ DOM дёҺ Diff з®—жі•зҡ„е®һзҺ°еҖҹйүҙдәҶ snabbdom иҝҷдёӘеә“пјҢsnabbdom жҳҜдёҖдёӘиҷҡжӢҹ DOM еә“пјҢе®ғдё“жіЁдәҺз®ҖеҚ•пјҢжЁЎеқ—еҢ–пјҢејәеӨ§зҡ„еҠҹиғҪе’ҢжҖ§иғҪгҖӮиҰҒеҪ»еә•жҳҺзҷҪиҷҡжӢҹ DOM дёҺ Diff з®—жі•е°ұеҫ—еҲҶжһҗ snabbdom иҝҷдёӘеә“еҲ°еә•еҒҡдәҶд»Җд№Ҳпјҹ

иҺ·еҸ–жәҗд»Јз Ғ

еҸҜд»ҘйҖҡиҝҮnpm i snabbdom -D жқҘдёӢиҪҪ snabbdom еә“пјҢиҝҷж ·жҲ‘д»¬ж—ўиғҪзңӢеҲ° src дёӢз”Ё Typescript зј–еҶҷзҡ„жәҗз ҒпјҢд№ҹиғҪзңӢеҲ°зј–иҜ‘еҘҪзҡ„ JavaScript д»Јз ҒгҖӮдёӢйқўиҙҙзҡ„жәҗз ҒжҳҜ 2.1.0 зүҲжң¬пјҢзҺ°еңЁе·Із»Ҹжӣҙж–°еҲ° 3.0.3 зүҲжң¬дәҶгҖӮе»әи®®е°ҶдёӢж–№еҮәзҺ°зҡ„жәҗз ҒеӨҚеҲ¶еҲ°дёӢиҪҪзҡ„ snabbdom еә“дёӯзӣёеә”дҪҚзҪ®пјҢиҝҷж ·зңӢжәҗз ҒжҜ”иҫғжё…жҷ°гҖӮйӮЈжҲ‘д»¬е°ұејҖе§ӢеҲҶжһҗжәҗз Ғеҗ§гҖӮ

жәҗз ҒеҲҶжһҗ

JavaScript еҜ№иұЎжЁЎжӢҹзңҹе®һ DOM ж ‘

йҖҡиҝҮи°ғз”Ё snabbdom еә“дёӯзҡ„ hеҮҪж•°е°ұеҸҜд»ҘеҜ№зңҹе®һ DOM иҠӮзӮ№иҝӣиЎҢжҠҪиұЎгҖӮжҲ‘д»¬е…ҲжқҘзңӢзңӢдёҖдёӘе®Ңж•ҙзҡ„иҷҡжӢҹ DOM иҠӮзӮ№пјҲvnodeпјүжҳҜд»Җд№Ҳж ·зҡ„пјҡ

е®һйҷ…дёҠпјҢhеҮҪж•°зҡ„дҪңз”Ёе°ұжҳҜз”Ё JavaScript еҜ№иұЎжЁЎжӢҹзңҹе®һзҡ„ DOM ж ‘пјҢеҜ№зңҹе®һ DOM ж ‘иҝӣиЎҢжҠҪиұЎгҖӮи°ғз”Ё hеҮҪж•°е°ұиғҪеҫ—еҲ°з”ұ vnode з»„жҲҗзҡ„иҷҡжӢҹ DOM ж ‘гҖӮ

и°ғз”Ё hеҮҪж•°жңүеӨҡз§ҚеҪўејҸпјҡ

дҪҝеҫ— hеҮҪж•°зҡ„з¬¬дәҢе’Ңз¬¬дёүдёӘеҸӮж•°жҜ”иҫғзҒөжҙ»пјҢиҰҒеҲӨж–ӯзҡ„жғ…еҶөд№ҹжҜ”иҫғеӨҡпјҢдёӢйқўжҠҠиҝҷйғЁеҲҶзҡ„ж ёеҝғжәҗз ҒеҲҶжһҗиҙҙдёҖдёӢпјҡ

diff з®—жі•вҖ“е…ҘеҸЈеҮҪж•°

йҖҡиҝҮ hеҮҪж•°еҫ—еҲ°ж–°ж—§иҷҡжӢҹиҠӮзӮ№ DOM еҜ№иұЎеҗҺе°ұеҸҜд»ҘиҝӣиЎҢе·®ејӮжҜ”иҫғдәҶгҖӮеңЁе®һйҷ…дҪҝз”ЁиҝҮзЁӢдёӯпјҢжҲ‘д»¬жҳҜзӣҙжҺҘи°ғз”Ё snabbdom зҡ„ patch еҮҪж•°пјҢз„¶еҗҺдј е…ҘдёӨдёӘеҸӮж•°пјҢйҖҡиҝҮ patch еҮҪж•°еҶ…йғЁеӨ„зҗҶе°ұеҸҜд»Ҙеҫ—еҲ°ж–°ж—§иҷҡжӢҹиҠӮзӮ№ DOM еҜ№иұЎзҡ„е·®ејӮпјҢе№¶е°Ҷе·®ејӮйғЁеҲҶжӣҙж–°еҲ°зңҹжӯЈзҡ„ DOM ж ‘дёҠгҖӮ

йҰ–е…ҲпјҢpatch еҮҪж•°дјҡеҺ»еҲӨж–ӯ oldVnode жҳҜеҗҰжҳҜзңҹе®һDOMиҠӮзӮ№пјҢеҰӮжһңжҳҜеҲҷйңҖиҰҒе…ҲиҪ¬жҚўдёәиҷҡжӢҹDOMиҠӮзӮ№ oldVnode = emptyNodeAt(oldVnode) пјӣз„¶еҗҺеҺ»жҜ”иҫғж–°ж—§ vnode жҳҜеҗҰжҳҜеҗҢдёҖдёӘиҠӮзӮ№ sameVnode(oldVnode, vnode)пјҢеҰӮжһңжҳҜеҗҢдёҖиҠӮзӮ№еҲҷзІҫзЎ®жҜ”иҫғж–°ж—§ vnode patchVnode(oldVnode, vnode, insertedVnodeQueue) пјҢеҰӮжһңдёҚжҳҜеҲҷзӣҙжҺҘеҲӣе»әж–° vnode еҜ№еә”зҡ„зңҹе®һ DOM иҠӮзӮ№ createElm(vnode, insertedVnodeQueue)пјҢеңЁ createElm еҮҪж•°дёӯеҲӣе»әж–° vnode зҡ„зңҹе®һ DOM иҠӮзӮ№д»ҘеҸҠе®ғеҜ№еә”зҡ„еӯҗиҠӮзӮ№пјҢе№¶жҠҠеӯҗиҠӮзӮ№жҸ’е…ҘеҲ°зӣёеә”дҪҚзҪ®гҖӮеҰӮжһң oldVnode.elm жңүзҲ¶иҠӮзӮ№еҲҷжҠҠж–° vnode еҜ№еә”зҡ„зңҹе®һ DOM иҠӮзӮ№дҪңдёәеӯҗиҠӮзӮ№жҸ’е…ҘеҲ°зӣёеә”дҪҚзҪ®пјҢе№¶дё”еҲ йҷӨж—§иҠӮзӮ№гҖӮдёӢйқўиҙҙдёҖдёӢ patch еҮҪж•°зҡ„жәҗз Ғи§Јжһҗпјҡ

patch еҮҪж•°дёӯз”ЁеҲ°дәҶ emptyNodeAt еҮҪж•°пјҢиҝҷдёӘеҮҪж•°дё»иҰҒжҳҜеӨ„зҗҶ patch еҮҪж•°з¬¬дёҖдёӘеҸӮж•°дёәзңҹе®һDOMиҠӮзӮ№зҡ„жғ…еҶөгҖӮдёӢйқўиҙҙдёҖдёӢиҝҷдёӘеҮҪж•°зҡ„жәҗз Ғи§Јжһҗпјҡ

patch еҮҪж•°дёӯз”ЁеҲ°дәҶ sameVnode еҮҪж•°пјҢиҝҷдёӘеҮҪж•°дё»иҰҒз”ЁжқҘжҜ”иҫғдёӨдёӘиҷҡжӢҹDOMиҠӮзӮ№жҳҜеҗҰжҳҜеҗҢдёҖдёӘиҷҡжӢҹиҠӮзӮ№гҖӮдёӢйқўиҙҙдёҖдёӢиҝҷдёӘеҮҪж•°зҡ„жәҗз ҒеҲҶжһҗпјҡ

diff з®—жі•вҖ“ж–°ж—§ vnode дёҚжҳҜеҗҢдёҖдёӘиҠӮзӮ№зҡ„жғ…еҶө

ж №жҚ® sameVnode еҮҪж•°иҝ”еӣһзҡ„з»“жһңпјҢж–°ж—§ vnode дёҚжҳҜеҗҢдёҖдёӘиҷҡжӢҹиҠӮзӮ№гҖӮйҰ–е…ҲиҺ·еҸ–еҲ° oldVnode еҜ№еә”зңҹе®һ DOM иҠӮзӮ№зҡ„зҲ¶иҠӮзӮ№ parent пјҢз„¶еҗҺи°ғз”Ё createElm еҮҪж•°еҺ»еҲӣе»ә vnode еҜ№еә”зҡ„зңҹе®һ DOM иҠӮзӮ№д»ҘеҸҠе®ғзҡ„еӯҗиҠӮзӮ№е’Ңж ҮзӯҫеұһжҖ§зӯүзӯүгҖӮеҲӨж–ӯжҳҜеҗҰжңү parentпјҢ еҰӮжһңжңүеҲҷе°Ҷ vnode.elm еҜ№еә”зҡ„DOMиҠӮзӮ№дҪңдёәеӯҗиҠӮзӮ№жҸ’е…ҘеҲ° parent иҠӮзӮ№дёӢзҡ„зӣёеә”дҪҚзҪ®гҖӮйғЁеҲҶжәҗз ҒеҲҶжһҗеңЁpatchеҮҪж•°дёӯпјҢдёӢйқўиҙҙдёҖдёӢ createElm еҮҪж•°зҡ„жәҗз ҒеҲҶжһҗпјҡ

diff з®—жі•вҖ“ж–°ж—§ vnode жҳҜеҗҢдёҖдёӘиҠӮзӮ№зҡ„жғ…еҶө

дёҠйқўеҲҶжһҗдәҶж–°ж—§ vnode дёҚжҳҜеҗҢдёҖдёӘиҷҡжӢҹиҠӮзӮ№пјҢйӮЈд№ҲжҳҜеҗҢдёҖдёӘиҷҡжӢҹиҠӮзӮ№еҸҲиҜҘжҖҺд№ҲеҺ»еӨ„зҗҶпјҹйҰ–е…ҲпјҢи°ғз”Ё patchVnode еҮҪж•° patchVnode(oldVnode, vnode, insertedVnodeQueue)пјҢиҝҷдёӘеҮҪж•°дјҡеҜ№ж–°ж—§ vnode иҝӣиЎҢзІҫзЎ®жҜ”иҫғпјҡ

в‘  еҰӮжһңж–°ж—§иҷҡжӢҹ DOM еҜ№иұЎе…Ёзӯү oldVnode === vnode пјҢйӮЈд№ҲдёҚеҒҡд»»дҪ•ж“ҚдҪңпјҢзӣҙжҺҘиҝ”еӣһпјӣ

в‘Ў з„¶еҗҺеҲӨж–ӯ vnode жҳҜеҗҰжңүж–Үжң¬иҠӮзӮ№ isUndef(vnode.text) пјҢеҰӮжһңжІЎжңүж–Үжң¬иҠӮзӮ№еҲҷеҲӨж–ӯ oldVnode дёҺ vnode жңүжІЎжңүеӯҗиҠӮзӮ№ isDef(oldCh) && isDef(ch)пјҢеҰӮжһңйғҪжңүеӯҗиҠӮзӮ№дё”дёҚзӣёзӯүеҲҷи°ғз”Ё updateChildren еҮҪж•°еҺ»жӣҙж–°еӯҗиҠӮзӮ№пјӣ

в‘ў еҰӮжһңеҸӘжңү vnode жңүеӯҗиҠӮзӮ№иҖҢ oldVnode жңүж–Үжң¬иҠӮзӮ№жҲ–жІЎжңүеҶ…е®№пјҢе°Ҷ oldVnode зҡ„ж–Үжң¬иҠӮзӮ№зҪ®з©әжҲ–дёҚеҒҡеӨ„зҗҶпјҢи°ғз”Ё addVnodes еҮҪж•°е°Ҷ vnode зҡ„еӯҗиҠӮзӮ№еҲӣе»әеҮәеҜ№еә”зҡ„зңҹе®һDOMе№¶еҫӘзҺҜжҸ’е…ҘеҲ°зҲ¶иҠӮзӮ№дёӢпјӣ

в‘Ј еҰӮжһңеҸӘжңү oldVnode жңүеӯҗиҠӮзӮ№иҖҢ vnode жІЎжңүеҶ…е®№пјҢеҲҷзӣҙжҺҘеҲ йҷӨ oldVnode дёӢзҡ„еӯҗиҠӮзӮ№пјӣ

в‘Ө еҰӮжһңеҸӘжңү oldVnode жңүж–Үжң¬иҠӮзӮ№иҖҢ vnode жІЎжңүеҶ…е®№пјҢеҲҷе°Ҷ oldVnode еҜ№еә”зҡ„зңҹе®һ DOM иҠӮзӮ№зҡ„ж–Үжң¬зҪ®з©ә;

в‘Ҙ еҰӮжһң vnode жңүж–Үжң¬иҠӮзӮ№пјҢoldVnode жңүеӯҗиҠӮзӮ№е°ұе°ҶеҜ№еә”зңҹе®һ DOM иҠӮзӮ№зҡ„еӯҗиҠӮзӮ№еҲ йҷӨпјҢжІЎжңүе°ұдёҚеӨ„зҗҶпјҢз„¶еҗҺе°Ҷ vnode зҡ„ж–Үжң¬иҠӮзӮ№дҪңдёәеӯҗиҠӮзӮ№жҸ’е…ҘеҲ°еҜ№еә”зңҹе®һ DOM иҠӮзӮ№дёӢгҖӮ

йғЁеҲҶжәҗз ҒеҲҶжһҗеңЁpatchеҮҪж•°дёӯпјҢдёӢйқўиҙҙдёҖдёӢ patchVnode еҮҪж•°зҡ„жәҗз ҒеҲҶжһҗпјҡ

diff з®—жі•вҖ“ж–°ж—§ vnode еӯҗиҠӮзӮ№зҡ„жӣҙж–°зӯ–з•Ҙ

еҪ“ж–°ж—§ vnode йғҪжңүеӯҗиҠӮзӮ№ж—¶пјҢdiff з®—жі•е®ҡд№үдәҶеӣӣдёӘжҢҮй’ҲжқҘеӨ„зҗҶеӯҗиҠӮзӮ№пјҢеӣӣдёӘжҢҮй’ҲеҲҶеҲ«жҳҜпјҡoldStartVnode(ж—§еүҚvnode)/newStartVnode(ж–°еүҚvnode)/oldEndVnode(ж—§еҗҺvnode)/newEndVnode(ж–°еҗҺvnode) гҖӮиҝӣе…ҘеҫӘзҺҜдҪ“еҶ…еҗҺпјҢж–°ж—§ vnode зҡ„еӯҗиҠӮзӮ№дёӨдёӨжҜ”иҫғпјҢиҝҷйҮҢжҸҗдҫӣдәҶдёҖеҘ—жҜ”иҫғи§„еҲҷпјҢеҰӮдёӢеӣҫпјҡ

еҰӮжһңдёҠйқўеӣӣдёӘи§„еҲҷйғҪдёҚж»Ўи¶іпјҢе°Ҷ oldVnode зҡ„еӯҗиҠӮзӮ№д»Һж—§зҡ„еүҚзҙўеј• oldStartIdx еҲ°ж—§зҡ„еҗҺзҙўеј• oldEndIdx еҒҡдёҖдёӘ key дёҺеҜ№еә”дҪҚзҪ®еәҸеҸ·зҡ„жҳ е°„ oldKeyToIdx пјҢйҖҡиҝҮж–° vnode зҡ„ key еҺ»жүҫ oldKeyToIdx дёӯжҳҜеҗҰжңүеҜ№еә”зҡ„зҙўеј•еҖјпјҢиӢҘжІЎжңүпјҢиЎЁжҳҺ oldVnode жІЎжңүеҜ№еә”зҡ„ж—§иҠӮзӮ№пјҢжҳҜдёҖдёӘж–°еўһзҡ„иҠӮзӮ№пјҢиҝӣиЎҢжҸ’е…Ҙж“ҚдҪңпјӣиӢҘжңүпјҢиЎЁжҳҺ oldVnode жңүеҜ№еә”зҡ„ж—§иҠӮзӮ№пјҢдёҚжҳҜдёҖдёӘж–°еўһиҠӮзӮ№пјҢиҝӣиЎҢз§»еҠЁж“ҚдҪңгҖӮдёӢйқўиҙҙдёҖдёӢжәҗз Ғи§Јжһҗпјҡ
